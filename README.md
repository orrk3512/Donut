# Donut
Recreation of the floating donut gif. Made of ascci characters in a terminal, the characters are chosen to create shading, the donut should rotate anf hover up and down, as well as be lit from above.

Goals for the project:
- The first goal was to recreate the original floating rotating donut animation.
- The second goal was to make system modular enough that individual systems could be swapped out. For example to have a spinning rotaing cube instead, or too swap out how the lighting is done.

This branch has been updated since the hackathon, to see the original project go to the "original" branch.
